# Privacy Note

**Project:** Ethical, Legal, IP, Professional Issues  
**Date:** 2026  
**Version:** 1.0

---

## 1. Overview

This document describes how this project handles personal data and user privacy.

---

## 2. Data We Collect

This project, in its current form as an academic final project, does **not** collect, store, or transmit personal data from users.

If future versions include user-facing features, this policy will be updated accordingly.

---

## 3. Data We Do Not Collect

We do **not** collect:
- Names, email addresses, or contact information
- Location data
- Browsing history or behavioral tracking data
- Sensitive personal information of any kind

---

## 4. Third-Party Services

If the project uses third-party libraries or APIs, those services may have their own privacy policies. Users are encouraged to review:
- The privacy policies of any third-party tools listed in `docs/ip-and-attribution.md`
- The terms of service of any external APIs integrated into the project

---

## 5. Data Security

Since no personal data is collected, no data storage or encryption infrastructure is required at this stage. Should this change, appropriate security measures will be implemented and documented.

---

## 6. User Rights

Users have the right to:
- Know what data is collected (none, at this time)
- Request deletion of any data (not applicable at this time)
- Opt out of data collection (not applicable at this time)

---

## 7. Contact

For privacy-related questions or concerns regarding this project, please open an issue in the project repository.
